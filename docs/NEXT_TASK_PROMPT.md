# Prompt Siap Pakai untuk AI Coding Agent

Salin seluruh isi di bawah ini dan tempel ke Cursor/Codex/Claude Code (atau tool AI lain) di dalam repository project ini.

---

Baca file `AGENTS.md` di root repo ini dan ikuti seluruh instruksinya.

Konteks tambahan: website untuk project ini sudah mulai dibangun sebelumnya (bukan dari nol). Sebelum mengerjakan task apa pun, lakukan langkah berikut secara berurutan:

1. Baca `docs/PRD.md`, `docs/BRD.md`, dan `docs/ROADMAP.md` untuk memahami keseluruhan rencana project.
2. Review struktur folder dan kode yang sudah ada di repo ini saat ini.
3. Bandingkan kode yang sudah ada dengan daftar task di `docs/ROADMAP.md`, tentukan task mana saja yang menurutmu sudah terpenuhi (walau belum tercatat), dan task mana yang belum dikerjakan sama sekali.
4. Buat atau update file `docs/PROGRESS.md` berisi kesimpulan review ini — tandai task yang sudah terpenuhi sebagai selesai (dengan catatan "disimpulkan dari review kode awal, bukan dikerjakan sesi ini"), dan tandai task pertama yang belum terpenuhi sebagai task berikutnya yang akan dikerjakan.
5. Tampilkan ringkasan hasil review ini kepadaku sebelum melanjutkan ke pengerjaan task — aku ingin konfirmasi dulu sebelum kamu mulai menulis/mengubah kode.

Setelah aku konfirmasi, lanjutkan mengerjakan satu task berikutnya sesuai urutan di `docs/ROADMAP.md`, ikuti seluruh aturan di `AGENTS.md`.
